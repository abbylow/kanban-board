## Kanban Board
A simple drag and drop kanban board with some CRUD features. Nothing special.

Utilize ```react-beautiful-dnd``` package to handle the drag and drop feature. 

### Feautures
- Rename board
- Create a list
- Rename a list
- Delete a list
- Drag and drop a list
- Create a card
- Edit a card
- Delete a card
- Drag and drop a card
